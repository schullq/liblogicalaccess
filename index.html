Open

Welcome to the LibLogicalAccess project web page. LibLogicalAccess is an open source RFID library developed in C++. It provides a good abstraction layer that lets you use differents cards and reader in a semantically consistent way.


Portable

LibLogicalAccess is designed for portablity. The library works well on both Linux and Windows, aswell as OSX. The codebase also build and works on both x86/amd64 and ARM.

Good hardware support

LibLogicalAccess architecture is modules based. This allows it to support supports various card readers and RFID chips.
